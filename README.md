# Med-E-Pal_repo
  Code base for Med-E-Pal Doctor Mobile App

# Pre-requiste:
  1. node / yarn - latest version
  2. git

# Installation steps:
  1. npm install expo-cli or yarn install expo-cli
  2. clone the package
  3. npm install or yarn install

# Run
  npm start or expo start or yarn start
